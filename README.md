# CountDrop - 残数管理アプリ

## 概要

iPhoneユーザー向けの残数管理アプリケーション。日用品やアイテムの数量を簡単に管理し、ウィジェットからも操作可能なカウンターアプリです。

## 技術スタック

- **フレームワーク**: Flutter v3.29.3
- **開発言語**: Dart 3.7.2
- **ストレージ**: shared_preferences v2.5.3
- **対象プラットフォーム**: iOS (iPhone)

## 主な機能

- アイテムの登録・編集・削除
- 残数のカウント管理
- ウィジェットによる簡単操作
- データのローカル保存

## セットアップ

1. 必要条件:
   - Flutter SDK
   - Xcode (iOS開発用)
   - iOS Simulator または iOS デバイス

2. インストール:
```sh
git clone [repository-url]
cd drop_counter
flutter pub get
```

3. 実行:
```sh
flutter run
```
